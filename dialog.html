Dialog class

class storytime_ai.dialog.Dialog(dialogid: str, text: str, choices: dict[str, Choice], logic: str = '')

Dialog class, that represents a single dialogue in the story.

A dialog provides markdown output and can contain logic in a basic syntax. This logic is parsed and used by the Story.exec_logic() method.

dialogid

The unique identifier of the dialog. It is a heading in the markdown output.

Type:

str

text

Dialog text. It is a paragraph in the markdown output. Markdown is supported, if the heading level is not higher than three (‘###’)

Type:

str

choices

A dictionary of Choice objects. The key is the next dialog id (heading), the value is the Choice object.

Type:

dict[str, Choice]

logic

A string containing the logic. It is parsed by the Story.exec_logic() method.

Type:

str
